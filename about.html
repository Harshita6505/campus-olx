About Campus OLX

This project is a campus-only resource exchange platform. It aims to let students post, discover, and exchange books, notes, equipment, and more.

Key goals

  • Campus-verified community
  • Simple posting and discovery
  • In-app messaging for safe negotiations

Built as a demo frontend

This website is a front-end demonstration — no backend. Forms and actions are simulated for demo purposes.
